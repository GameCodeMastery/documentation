## Features:

- Melee Combat System (Dual Wield, 2H, 1 Handed, Fists, 1 Hand Shield, Shield, twinblades, & more)
- Ranged Combat System
- Magic Combat System
- Multiple "Builds" (Melee, ranged, magic, multiple equipment types, abilities, buffs, multiple damage types, etc)
- Advanced Boss AI System
- Advanced Mob AI
- Complex & Detailed Hit Reaction System
- Directional Blocking
- Fatality Execution System
- Assassinations
- Parry System
- Abilities System
- Gameplay Effects System
- 8-way Directional Dodging
- Input Buffer System
- Targeting System
- Camera Manager
- Targeting System
- RPG attributes System
- RPG Equipment System
- RPG Inventory System
- Storage & Looting
- NPC Vendor Shop
- Inventory & Equipment UI
- Item Sorting System (sort by type, rarity, etc)
- Item Switching System (item switching UI included)
- Multiple weapon types
- Multiple damage types
- Status Effects System (poison, frostbite, etc)
- Impact Sounds & VFX System
- Interaction system
- Accessories System (grant various buffs or other attribute modifiers)
- Multiple Stances (player character)
- Turn in-place
- Save Game
- Gamepad Optimized
